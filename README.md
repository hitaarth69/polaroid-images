# Polaroid Images — Professional Layout Studio

A premium, browser-based tool for generating Polaroid-style photo layouts. 
Designed for photographers, designers, and print studios.

## Features
- Upload multiple images via drag & drop
- Choose from 3 layout styles: Classic Grid, Offset Artistic, Angled Random
- Customize frame size, image size, borders, shadows, captions, and spacing
- Support for A4, A3, 12×18, Letter, Legal, Tabloid, and custom page sizes
- Live real-time preview with zoom controls
- Drag and reposition individual Polaroid frames on canvas
- Export as SVG (CorelDRAW 2018 compatible), PNG, JPEG, or PDF
- Past Works gallery saved in browser local storage
- Dark and Light theme with smooth transitions
- Animated intro loader and custom cursor
- Fully offline — no backend, no frameworks, no dependencies

## How to Use
1. Open index.html in any modern browser
2. Upload your images
3. Choose your page size and layout settings
4. Click Generate Layout
5. Download in your preferred format
